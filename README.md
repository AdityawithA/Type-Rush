# ⌨️ TypeRush - Typing Speed Tester ⚡

TypeRush is a colorful and interactive typing speed tester built using pure HTML, CSS, and JavaScript.
It helps users improve their typing speed and accuracy through real-time feedback, multiple difficulty levels, and a modern neon-inspired UI.

---

# 🚀 Features

* ⚡ Real-time WPM calculation
* 🎯 Accuracy tracking
* ❌ Live mistake highlighting
* ⏱ Multiple timer modes (30s / 60s)
* 🧠 Difficulty levels (Easy / Medium / Hard)
* 🏆 Local score history using localStorage
* 🎨 Modern colorful neon UI
* 📱 Fully responsive design

---

# 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

No frameworks or libraries were used.

# 🎮 How to Use

1. Select difficulty level
2. Choose timer duration
3. Start typing the displayed text
4. View live:

   * WPM
   * Accuracy
   * Mistakes
   * Remaining time
5. Check your score history after completing the test

---

# 🧠 WPM Formula

```text
WPM = Total Words Typed / Time in Minutes
```
